Roles Module

This module allow you to create dynalic roles with an associated set of permissions and grant them to users (accounts or space ids) within a given space.

For example if you want to create a space that enables editors in a similar way to Medium, you would create a role “Editor” with permissions such as CreatePosts, UpdateAnyPost, and HideAnyComment. Then you would grant this role to the specific accounts you would like to make editors.
